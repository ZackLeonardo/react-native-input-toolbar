# react-native-input-toolbar

## Installation
The component which can be used as input tool bar.

## Usage
 (for myself in private npm store)
```
yarn add @zdy/react-native-input-toolbar
```
(for everyone)
```
yarn add file:/...path.../react-native-input-toolbar
```
